## IM21TS Relay Breakout PCB

Breakout board for TE Connectivity's IM21TS relay.  Converts to 0.1" breadboard format.

### Component Description

IM21TS Relay:
    Non-Latching, DPDT
    Coil Voltage - 3VDC
    Coil Current - 16.7mA
    Switch Voltage - 250VAC
    Contact Rating - 2A

Digi-Key part #PB1167-ND
